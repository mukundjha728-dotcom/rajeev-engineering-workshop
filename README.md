# Rajeev Engineering Workshop

A premium industrial fabrication business website built for **Rajeev Engineering Workshop**, showcasing **1000+ iron fabrication designs** including gates, shutters, railings, grills, sheds, furniture, fencing, and custom iron works.

## Live Website

🚀 Live Demo: https://rajeev-engineering-workshop-six.vercel.app/

---

## Overview

Rajeev Engineering Workshop is a Darbhanga-based iron fabrication business operating for **10+ years** with **5000+ satisfied customers**. This platform was built to digitize their business, display their large product catalog, and generate leads directly through calls and WhatsApp.

The website is designed with a premium industrial aesthetic using a dark steel navy + gold theme.

## Installation

Clone the repository:

```bash
git clone https://github.com/mukundjha728-dotcom/rajeev-engineering-workshop.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build production:

```bash
npm run build
```
